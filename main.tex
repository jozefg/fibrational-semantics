\title{Notes on Fibrational Semantics}
\author{Daniel Gratzer}
\date{\today}
\begin{document}
\begin{abstract}
  I have lately found myself reading a large number of papers which
  deal in some way with fibrations in category theory. Particularly to
  give semantics to a logic or type theory. In order to be sure that
  I've internalized these concepts in some small way I have written up
  some notes introducing fibrations.

  It would be far too ambitious to claim the completeness of these
  notes (this would be closer to a dissertation \cite{Jacobs:91}
  rather than a note). Rather, I shall aim for a level of completeness
  that let's me state and prove semantics for an interesting type
  theory. After that, I will gesture towards how the semantics might
  be extended for more featureful logics or type theories but avoid
  the same level of proof.
\end{abstract}
